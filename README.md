# pinyin-search

[KeeWeb](https://keeweb.info) 插件，为 KeeWeb 添加拼音搜索支持。

支持拼音全拼（如 `taobao` → 淘宝）和拼音首字母（如 `wx` → 微信）搜索标题、用户名、网址、备注和标签。

## 安装

KeeWeb → **设置** → **插件** → 从 URL 添加插件 → 输入下面的 URL → **安装**。

```
https://ramboo1990.github.io/keeweb-pinyin-search/
```

> 插件尚未发布到 GitHub Pages。当前只能在开发环境下测试，详见 [AGENTS.md](AGENTS.md)。

## 搜索示例

| 输入 | 匹配 |
|------|------|
| `zhifubao` | 支付宝 |
| `zfb` | 支付宝 |
| `alipay` | 支付宝 |
| `taobao` | 淘宝 |

非中文文本原样搜索，不受影响。
